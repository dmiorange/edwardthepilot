# Edwardthepilot

Personal website for **Edwardthepilot** — a Swedish commercial Airbus A220 pilot and Twitch flight sim streamer. Built with plain HTML, CSS, and JavaScript. No frameworks, no dependencies. Hosted on GitHub Pages.

🌐 **Live site:** [dmiorange.github.io/edwardthepilot](https://dmiorange.github.io/edwardthepilot)

---

## Pages

| Page | Description |
|---|---|
| **Home** | Full-viewport hero with live Twitch status indicator and links to stream and community |
| **About Me** | Bio, profile photo, tags, and IRL Flight Blog section with Discord channel embed and photo gallery |
| **MSFS** | Microsoft Flight Simulator settings for MSFS 2024 and 2020, virtual airlines (vSAS, vNorwegian), and scenery link |
| **Scenery** | Interactive Google My Maps embed showing favourite routes and destinations from the stream |
| **Hardware** | Full specs for the dual-PC setup — gaming rig and dedicated streaming PC, each with links to product pages |
| **Community** | Discord server link, the story behind the stream, and animated member count |
